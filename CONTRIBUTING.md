# Contributing

## Development

1. Install dependencies:
   - `npm install`
2. Run dev mode:
   - `npm run dev`
3. Run checks before pushing:
   - `npm run typecheck`
   - `npm run build`

## Pull Requests

1. Keep PRs focused and small.
2. Include a short summary of what changed and why.
3. Add test/verification notes (commands you ran).
4. Do not commit secrets, build artifacts, or local config.

## Commit Style (recommended)

- `feat: ...`
- `fix: ...`
- `chore: ...`
- `docs: ...`
- `refactor: ...`
